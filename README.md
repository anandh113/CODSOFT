# 🐍 Python Mini Projects Collection

A collection of beginner-friendly **Python projects** built to practice
programming logic, problem solving, and basic application development.

------------------------------------------------------------------------

## 📌 Projects Included

1.  📝 Todo List\
2.  🧮 Simple Calculator\
3.  🔐 Password Generator\
4.  ✊ Rock Paper Scissors Game\
5.  📇 Contacts Management System

------------------------------------------------------------------------

## 📝 1. Todo List

A simple command‑line task manager to organize daily tasks.

**Features** - Add tasks - View tasks - Mark tasks as completed - Delete
tasks

**Run**

``` bash
python todo_list.py
```

------------------------------------------------------------------------

## 🧮 2. Simple Calculator

A basic calculator that performs arithmetic operations.

**Features** - Addition - Subtraction - Multiplication - Division

**Run**

``` bash
python calculator.py
```

------------------------------------------------------------------------

## 🔐 3. Password Generator

Generates strong random passwords.

**Features** - Custom password length - Mix of letters, numbers, and
symbols

**Run**

``` bash
python password_generator.py
```

------------------------------------------------------------------------

## ✊ ✋ ✌ 4. Rock Paper Scissors Game

Play Rock Paper Scissors against the computer.

**Features** - Random computer choice - Instant result display

**Run**

``` bash
python rock_paper_scissors.py
```

------------------------------------------------------------------------

## 📇 5. Contacts Management System

Store and manage contact details.

**Features** - Add contacts - Edit contacts - Delete contacts - View
contact list

**Run**

``` bash
python contacts_manager.py
```

------------------------------------------------------------------------
